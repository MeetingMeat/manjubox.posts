## 概要
wavファイルを指定したサンプリングレートにアップサンプリングします。  
ダウンサンプリングには対応していません。

## 使い方
YukkuResampler.exeにリサンプリングしたいwavファイルをD&Dしてください。

## オプション
---
| 引数    | 概要 |
| ------- | --- |
| -h      | ヘルプ |
| -linear | Linear変換 |
| -cubic  | Cubic変換（デフォルト） |
| -48000  | 48kHzに変換（デフォルト） |
| -44100  | 44.1kHzに変換 |
| -192000 | 192kHzに変換 |
